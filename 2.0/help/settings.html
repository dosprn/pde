Program settings
Previous  Top  Next


To open the program settings, switch to projects editor and find the Settings option in the main menu. The Settings window has two tabs. On the first tab you can configure:

·Scale. Here you can configure the coordinate grid scale: by default, meters are used though you can change this value.  
·Width of the spacings to ignore. Configure the width of the spacings that program will ignore and consider accurately joined ends of the segments.  
·Color of tile borders. Select a color for drawing tile borders on plot.
