Known issues

1. You can come across the following issues while using Project Manager, Tiles Editor and/or Patterns Editor:
·Though a particular tool is activated, i.e. the tool button is pressed, it is not functioning. To solve the issue, please disable the tool and then enable it once more;  
·The program can fail to correctly switch to another tool: e.g. the info displayed in the status-window does not belong to the defined element but to the previous one; the selected element is outlined with the wrong line type and etc. Nevertheless, these issues do not influence the data integrity since these are visualization problems only. Just disable the tool and then press the button once again.  

2. You can come across the following issue while using Tiles Editor and/or Patterns Editor:
·The area calculating errors are possible if the arcs are present in the outline.  

3. You can come across the following issues while using Tiles Editor and/or Halves Editor:
·On selecting a half a broken polygon is displayed instead of accurate outline  
 

Possible reasons are as follows:
A. You have drawn a zero-length line segments while creating the outline. This may occur if you double click at the same place so that the start point and the endpoint of the outline will match. Please use the List of graphic objects to delete those segments.  
B. While selecting halves you have also selected elements that should not be present in the outline. To overcome this issue, please undo the selecting and try again.  

4. It may take some time to perform certain actions for slow computers. Do not try to repeat, undo or give another commands before a previous command is done since it may cause program failure.

If you come across the issues or errors that are not described here, please contact our support team.
