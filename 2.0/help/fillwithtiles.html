Paving the plot
Previous  Top  Next


To pave the plot, press Fill on the toolbar or go to Plot > Fill.
The program will pave the plot. The first step of this process is adding the pattern you have previously specified on the start point. Further patterns are put on the reference points specified for the neighboring patterns and so on. The paving process may take some time to complete. The program will notify you when the task is completed and you will be able to view the result:


What can go wrong?
1.You have not checked the pattern for spacings or put one tile on another.  
Edit the pattern, then select it once again and press .  

2.You have not placed tiles exactly along the side they should border with.  
Edit the reference point and press again.  

3.You have found empty wedges on the plot extending from one corner to another.  
You joined the outline inaccurately. Correct the outline and press .  

4.The program displays the dialog window notifying you that the plot is paved though it is empty.  
This means that you have chosen the wrong place for the reference point on the plot - neither the reference point of the first fragment nor reference points of the neighboring fragments got to the plot. A typical situation is when you put the reference point on the sharp corner of the plot. Set the reference point anywhere else.  
      
Now let us consider the second possible issue - when you have not placed tiles exactly along the side they should border with. This issue becomes apparent when you set the reference point of the pattern in the wrong place, e.g. inside of the pattern instead of setting it on the tile border. If you cannot calculate the exact place where the reference point should be set, try to experiment using the button. Press it to put a single fragment of the pattern on the reference point, then edit the reference point and try once again.

To delete all tiles at once, press .
If you need to re-pave the plot, it is not necessary to delete previous tiles: press Fill on the toolbar or Plot > Fill the program will automatically delete previous tiles before paving the new ones.
