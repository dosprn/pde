This Quick Start Guide covers the main features available in the Paving Design Expert, and will help you get started right away. It is not aimed to disclose all the niceties of the program usage. It gives you a general overview of the program: how to create a simple tile and how to fill a certain area with tiles. To learn more about any of the features outlined here, or to get guidelines please study the following chapters in the following order:
1.Quick start guide.  
2.Reference.  
3.Chapters containing practical advice on how to use the program: Tiles Editor, Patterns Editor, Developing Projects.  

Though you may immediately start reading the chapters containing practical info, the Reference will certainly be useful while studying this manual.

So, let's start.

After you have installed the program it's time to create a database containing:
1.Drafts of the tiles described on the Step 1.  
2.A set of patterns for each type of the tile described on the Step 2.  
Pattern- is the way you arrange tiles relative to each other.  

After the tiles and patterns database is created, you can start developing your projects. Every time you start a new project you will have to draw a plot that will be paved, choose the pattern, set the reference point and paving orientation. This is briefly described on the Step 3 of the Quick Start Guide.
