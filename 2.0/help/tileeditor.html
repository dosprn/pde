In order to start working with tiles you should create a new document. Go to Tiles > New or press on the toolbar.

The toolbar will be activated only after you create or reopen the tile.

A tile is drawn on the draft with the help of the following tools: Segment , Polyline and Arc . To learn how to draw in details, please refer to the Drawing graphic objects article. Refer to it for explanations. If you find any size errors, you can change the drawing entity as described in the Editing graphic objects article. The description of how to draw and edit the graphic objects is deliberately separated in the independent articles since the same methods are used when creating a plot. Though the main points will be mentioned here for better understanding of the drawing process in this chapter.

To draw a tile you can use two types of lines - solid and dotted ones. You can choose the line type you need in the drop-down menu on the toolbar:

It is important! The external outline should be drawn with a solid line. Dotted line is meant for drawing decorative ornaments on a tile. These line types don't have a great visual differences.  
Irrespective of what you start drawing first - outline with solid line or the decorative pattern of a tile (the second variant is less possible) - the first point set on the draft becomes reference point. It will be further used as a start point to rotate a tile, to create a pattern, and etc. The reference point is displayed on the draft as a purple cross.

Tile size is always shown in millimeters. This is done so to help you draw complex tiles more precisely, especially those with arcs. If the tile size is too large and does not fit display, feel free to zoom out using the button. Note that after you zoom out the scale it may become harder to draw the outline. For more details about the draft scaling, view the Scale article.

Advice. In the Drawing graphic objects article you can find the description of how to work with mini-display if the scale of a daft is too large.

If a tile does not fit draft then draft's size may be changed.

Along with drawing an outline of a tile, the program tries to calculate its area and the number of tiles needed to fill a square meter. This info is displayed on the toolbar.

To ensure the result is correct, left click on the statistics to make the program re-calculate the figures. Along with the tile area and number of tiles per square meter, the program also displays its thickness. By default it is equal to zero. To set the tile thickness, go to Draft > Tile Thickness.

Using this menu you can also comment on the tile created. Comments are typically used to note any additional info or remarks.

After you are done, save the tile draft: go to Tiles > Save or press on the toolbar. Then click Yes on the confirmation window.

By default, a new tile will be named "Untitled":

If you click OK right away, then the tile will be added in the list of tiles under the default name:

Name the newly created tile.After the name is assigned, you can specify the tiles set, where the new one should be added.

After save:


Some tiles are supplied as a set. As a rule, it's no sense to create a pattern using only one type of the tile from the set. The tiles of a set are put together to create a pattern with the help of several tile types. That is why we advise you to save such tiles as a separate set. Even single tiles (those types that can form pattern without other tile types) can be grouped by a certain character, e.g. colored, smooth, and etc. To create a new set, click New set. Name the set in the pop-up window:

Then name the tile and add it to the newly created set:

If you create more tiles that conform to the set, do not forget to specify that set to add new tile when you save them.

Advice: If you added a tile to the general list or wrong set by mistake, open this draft, save it once again using the Tiles > Save as... option. After you add the tile to the right set, open the wrong draft and delete it.


When you save the tile, the program also saves parameters of the draft and scale. When you open this draft next time to view or edit it, the position of tile will be the same when saved the draft or closed it last time.

To edit the tile, go to Tiles > Open or press on the toolbar. Select the tile you need in the list and press OK:

If you tick the Preview checkbox, you will be able to preview the tile, before opening it:

Select the tile and press OK to open it in the editor. Now you can modify it, save under another name, change name or even delete it.
DO NOT FORGET that if you delete a tile, you will also delete all its patterns and halves. If you are sure that you do not need the opened tile any more, go to Tiles > Delete to delete it:


Using this menu you can also export the tile draft to the BMP file. The Export to bmp option can be found above the Delete option in the Tiles menu. Enter the image name in the "Save as..." window. After you press Save, the 64x64 pixels image will be saved on your hard disk.

If the tiles set has also halves, you should draw them in the same document to be able to export them too. View the next article for details.

After the new tile is ready, create at least one pattern using it. To open the Patterns editor, switch to the main window and find it in the menu. You can also click on the toolbar to open the editor. This button will be available only after you save the draft.
