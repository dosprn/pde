System requirements
Previous  Top  Next


The minimum system requirements are as follows:
1. Processor: Pentium/Celeron class computer 200 or higher;
2. OS: Windows 98 or above;
3. Video: 800 x 600 pixels and 16-bit color as minimum;
4. Memory: min 64 MB;
5. Hard Disk space: 5 MB for installation and from 5 KB to 1-2 MB for each project.
