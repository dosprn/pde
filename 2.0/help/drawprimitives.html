Drawing graphic objects
Previous  Top  Next


With the help of the program you can draw 3 types of the graphic objects: line segments, polylines and arcs. These graphic objects are used in the Tiles editor and Developing the projects and are present on their toolbars: Segment , Polyline and Arc . Note that only one tool can be enabled at a time.

Drawing segment lines
To draw a segment line, press Segment on the toolbar, then:
·Move the pointer to the point on the draft where you will start drawing the segment line;  
·Left click to set the starting point of the segment line;  
·Move the pointer to the point on the coordinate grid where the segment will end: the segment will be displayed via dotted line;  
·Click left once again to set the endpoint of the segment line.  

Drawing polylines
To draw a polyline, press the Polyline button on the toolbar, then:
·Move the pointer to the point on the draft where you will start drawing polyline;  
·Left click to set the starting point of the segment line;  
·Move the pointer to the endpoint of the first segment on the coordinate grid: the segment will be displayed via dotted line;  
·Left click once again to set the endpoint of the second segment and start drawing the third one;  
·Repeat the sequence as many times as you need to draw a tile or plot;  
·After you are done, click right to disable the tool.  

Drawing arcs
To draw an arc, press the Arc button on the toolbar, then:
·Move the pointer to the imaginary centre of circle the arc belongs to;  
·Click left to set the centre of the geometric figure;  
·Move the pointer to the point where the arc will start; the outline of the arc will be displayed via dotted circle;  
·Click left to set the start point of the arc;  
·Move the pointer to the endpoint of the arc: the arc you will get is displayed with a solid line (anti-clockwise). Note that you cannot change the radius of the arc on this step;  
·Click once again to finish drawing the object.  

If you click right while setting the points of a segment or arc, you will reset the geometric figure you have already drawn.

If you press and hold Ctrl the, the pointer will move randomly, not rounded off to the divisions of the coordinate grid: you may need to start drawing segments not only on the lattice junctions but also inside cells.

When drawing any geometric object, join the segment endings very accurately: it is more convenient when the pointer motions are tied to the junctions of the coordinate grid (view Draft for details) and the scale is large. The large scale will also help you accurately join segments endings in the cell, for example in the point with 10,73; 11,8 coordinates.

The more accurate you are while joining the segments, the more precise data on the plot size you will get, and the more accurate paving of the plot will be. It is acceptable that the endings of the drawing entities (segments) are not always perfectly joined: the program will ignore 10 mm spacings for tiles and 50 mm spacings for a plot. You can easily modify these values in the program settings.

You can come across the situation when you need to draw a big object that does not fit the display: you have to use a small scale but what to join segments accurately. In this case you can use a mini-display You can also use mini-display to quickly navigate on the draft (not to scroll every time you need to reach a distant point). For example:
·Set the start point of the line segment;  
·Choose the point on the mini- display where you want to set the endpoint of the segment and click it: you will be automatically redirected to this area of the draft;  
·Set the endpoint on the draft.
