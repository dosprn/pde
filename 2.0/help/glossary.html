Reference point
Reference point is the basic point of the tile, pattern and a plot. When it concerns a tile and/or a plot, the reference point is a basic point of the graphic object that was drawn first. When it concerns a pattern, the reference point specifies where the first tile will be put. Any rotation (i.e. the change of the objects orientation) is made around the reference point. If you delete the first graphic object (of the tile or plot) or the first tile (of the pattern) but other graphic objects are still present, then the reference point will be the same. The reference point can be deleted only when all graphic objects (of the tile or plot) or tiles (of the pattern) are deleted.

Pattern
A pattern means a specific arrangement of the tiles while paving the plot (the color of the tiles is not taken into account). The tiles' coloring is specified after the tiles are paved. As you see, a pattern is a solely the way tiles are arranged relatively each other during the paving process. To set the pattern, draw its fragment first. When paving the plot, the program will automatically put this fragment on the plot and try to multiply it to fill in the plot: the neighboring fragments are put on the points specified when the pattern was drawn. For more details, please view the Patterns editor article.

Graphic objects, drawing entities
This term is borrowed from the computer graphics. Here it means a line segment or a circular arc.
