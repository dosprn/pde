Developing the project
Previous  Top  Next


The main window that you see on the program start is aimed to develop projects. You can either start a new project (Projects > New) or open the existing one (Projects > Open). After you open the document, all tools available on the toolbar will be enabled.


Working with a project comprises the following steps:
1.Drawing the plot  
2.Setting the reference point  
3.Selecting the pattern  
4.Paving the plot. First, you can try your hand with paving only one pattern, if needed  
5.Decorating the plot with color and texture  
6.Printing the results  
7.Saving the project
