If the tiles set contains forms for creating halves, then you can do the same with the help of the program to later use them in the patterns. To create halves divide the existing tile using the toolbar. Click the Halves Editor button to start. The button will be available only after you save the tile.


Below you can view the window of the editor:

In the left section you will find the list of halves sets; in the right section you will find the list of the halves present in the set. For example, "Vertical section" is the set of halves, "Left" and "Right" are the halves included in this set.

First, add the set. Click Add in the left section of the editor. Let us name it "Vertical section".


Now draw a line that will divide the tile upright. The line is drawn with the help of the tools used to create the tile outline. The line that will divide the tile should be solid. The line itself is colored blue:

You can edited the line the same way other graphic objects are edited.

The next step is to select the halves. This is done with the help of the special tool, i.e. press the button and then select opposite parts of the tile to create halves:
·Left click and hold the button to start selecting the first half;  
·   Move the pointer to form a rectangle. Make sure that only the half you select is fully captured in the rectangle;
·Release the left button.  
 
If everything is done correctly, the half will be added to the list, in the right section of the Halves window named "Untitled".

If you select the name of the half in the Halves window, it will be singled out on the draft with the help of the green background:


Important. If you get a broken polygon instead of an accurate half, then:


Possible reasons:
1. You have drawn a zero-length line segments while creating the outline. at the same place so that the start point and the endpoint of the outline will match. Please use the Editing graphic objects to delete those segments.
2. While selecting halves you have also selected elements that should not be present in the outline. To overcome this issue, please undo the selecting and try again.

A few words should be said about the halves names. In the example above, both halves are named "Undefined". This is possible only because both halves are identical geometrical figures, i.e. they are interchangeable if to pave the plot. To confirm their interchangeable status, tick the Interchangeable checkbox. These halves will not be displayed as separate figures in the report but will be presented as "Vertical section".

The halves of the tile above are not interchangeable: you should not check the "Interchangeable" box and the halves should be named differently, e.g. "Left" and "Right", so that you can easily identify them in the report.


Other buttons are:

On the right:
·Edit - click to edit the name of a half;
