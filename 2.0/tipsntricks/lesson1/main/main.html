This is how working project looks like after creating the edgings.

Now we have to draw the boundaries of the internal space. On the picture below they are shown in blue.

There are several ways to draw the borders in these conditions.
  1. In this lesson we have already seen how to draw using the line drawing dialog.
  2. Just draw the line, as described in the reference manual. In this case we will have to trace the outline of the layout. For this purpose we need to accurately hit the mouse to the reference points, which is not easy to do at a small scale. In a large scale the ends of the reference line may be beyond the screen and drawing a line from end to end will be impossible.
  3. In order to solve the problem mentioned above it is convenient to draw using the mini display.
So we create a new project. Draw the contours by one of the depicted ways, save a project named "Inner space" and choose a pattern for it "Crosses 100x200 (simple pattern) ". Pattern will appear in the upper left corner of the draft.

Set the starting point in the lower left corner and fill the tiles .

Now you need to paint the tiles. Pre-read how to create and save custom colors for later use. Suppose colors have been created, as it is written. We need to load them into the project.

But let's look more closely. Fragments in the even rows are not the same fragments as in the odd rows. Here tiles are stacked in different ways. So you won’t spread the color pattern on all the fragments.

Now we are going to remake it. At first you need to clear the tiles from the coloring. There are two ways to do it:
  1. Radical. Remove all the tiles and complete the project again.
  2. Assign the default gray color to all tiles.
Let’s consider the second method. Choose the tool "Select tile" and draw out all the tiles with the mouse while holding the left mouse button:

When we release the mouse button in the right-bottom corner, all the tiles will be selected:
Remember that active project we have now is "Inner space." Therefore, its tiles were selected only, although we covered everything by mouse including edging tiles.
Open the color select dialog and choose the tiles color by default.
